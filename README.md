#### 使用说明

1.  下载源码
2.  安装nodejs
3.  npm install 执行代码安装 
4.  hexo g 生成静态文件
5.  hexo s 启动验证
6.  hexo d 推送部署

#### 升级步骤

```
npm i hexo-cli -g
npm install -g npm-check
npm install -g npm-upgrade
npm-check
npm-upgrade
npm update
```

​	升级hexo, 可能不好使 那就升级主题再试试

​	升级主题: 重新修改配置和语言 用上个版本文件夹里的2个配置文件(和原生未修改的)对比,重新一个一个修改,而不是对比新旧2个配置文件
